# Treinamentos em PDF

Coloque os arquivos PDF de treinamento nesta pasta para que os operadores possam visualiza-los na Central da Qualidade.

## Como adicionar treinamentos

1. Salve seu arquivo PDF nesta pasta (`public/presentations/slides/`)
2. O sistema detectara automaticamente o arquivo
3. Os operadores poderao visualizar o PDF diretamente na aba "Treinamentos"

## Estrutura

```
public/presentations/slides/
  ├── README.md (este arquivo)
  ├── Treinamento-Inicial.pdf
  ├── Procedimentos-Atendimento.pdf
  └── Manual-Qualidade.pdf
```

## Dicas

- Use nomes descritivos para os arquivos (ex: `Treinamento-Caixa.pdf`)
- Os PDFs serao exibidos diretamente na interface, sem necessidade de download
- O titulo exibido sera o nome do arquivo sem a extensao .pdf
